# 材料
- 豚こま切れ肉
　 160g
- じゃがいも
　 １個
- 玉ねぎ
　 1/2個
- 万能ねぎの小口切り
　 ２本分
- 温かいご飯
　 どんぶり２杯分
- 煮汁
　 だし汁
　　 １カップ
　 しょうゆ
　　 大さじ２　1/2〜３
　 砂糖
　　 大さじ２
　 みりん
　　 小さじ１

# 作り方
- 1. じゃがいもは５mm厚さのいちょう切りに、玉ねぎは縦５mm幅に切る。
- 2. 鍋に煮汁の材料とじゃがいもを入れて火にかけ、じゃがいもがやわらかくなったら玉ねぎ、豚肉を１枚ずつ広げながら加える。じゃがいもが少々煮くずれ、玉ねぎが色よく煮えたら火を止める。
- 3. 器にご飯を盛り、２をのせる。つゆもたっぷりかけ、仕上げに万能ねぎを散らす。
